Консольное клиент–серверное приложение на C++ для передачи сообщений между процессами через бинарный файл (FIFO).
Основные компоненты:
Receiver: процесс, принимающий сообщения из кольцевой очереди; создаёт файл, запускает Sender и обрабатывает команды read/finish.
Sender: процесс, отправляющий сообщения в очередь; получает имя файла через аргументы командной строки, блокируется при переполнении.
MessageManager: модуль для записи/чтения сообщений, реализует кольцевую очередь и управляет заголовком очереди (head, tail, capacity).

Синхронизация:
Mutex — взаимное исключение при работе с файлом.
Semaphore — контроль заполненности и свободного пространства очереди (hFull, hEmpty).

Технологии и инструменты:
C++17, стандартные контейнеры и типы.

WinAPI: CreateProcess, WaitForSingleObject, CreateMutex, CreateSemaphore, работа с fstream в бинарном режиме.

Обработка ошибок:
Проверка открытия и корректности файла.
Контроль переполнения и пустой очереди.
Защита от некорректных команд и пустых сообщений.

Тестирование:
GoogleTest для проверки порядка FIFO, корректности записи/чтения и работы синхронизации.
